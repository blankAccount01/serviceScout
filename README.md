# About Service Scout

A simple browser extension that lets you ping websites and see if they're up at a moment's notice. Works with websites URLs and IPv4 addresses.  Use the top search bar to ping a website and store up to 3 websites to automatically ping when you open the extension. Taking a break between long projects ðŸ™‚.
